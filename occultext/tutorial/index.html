The Occultext Tutorial

Creating Occultext

Enter a word and hit "Viel it!"; an encrypted text will show up below.

Decrypting Occultext

You can decrypt Occultext by entering an Occultext string in the Uncover section; the Decrypted Occultext will show up below.

Secret Keys

You can add optional secret keys to your Occultext; the Occultext cannot be decrypted without the correct secret key.

Occultext with Markdown, Javascript and HTML Support

Occultext supports rendering an HTML, Javascript, Markdown, and possibly CSS Codes (this requires a basic web development skills)


Sample code lol:



    
Hello world

Please refer to this documentation for a better understanding of Markdown.

Images

To add images to your Occultext, you can use HTML image tag, below is an example:

<img src="link or path to the image">

More info can be found here.

Background Audio

To add audio, you can use the HTML audio tag:

<audio controls autoplay>
    <source src="horse.ogg" type="audio/ogg">
    <source src="horse.mp3" type="audio/mpeg">
    Your browser does not support the audio element.
  </audio>

For YouTube embed background autoplay:

<iframe style="height:1pwidth:1px;opacity: 0;" width="560" height="315"
    src="https://www.youtube.com/embed/VIDEO_ID?autoplay=1" frameborder="0" allow="autoplay; encrypted-media" allowfullscreen></iframe>

More info about audio can be found here.
